# PPGN Reproduction

## Requirements

Please refer to [PPGN](https://github.com/hadarser/ProvablyPowerfulGraphNetworks_torch)

## Usages

unzip dataset

```bash
unzip brec_v3.zip
```

To reproduce results on PPGN, run:

```bash
python test_BREC_search.py
```

The configs are stored in configs/BREC.json
